Kubernetes: get service account secret for dashboard login

by adminPosted on2018-04-10

To login to the Kubernetes dashboard, you need a token. This token can be retrieved with the following command:

kubectl -n kube-system describe secret $(kubectl -n kube-system get secret | grep  | awk '{print $1}')
